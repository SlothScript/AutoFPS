# AutoFPS
A simple WoW addon that creates and positions your FPS counter

## Capabilities
* FPS Counter - Shows your average FPS over a few seconds
* FPS Relocation - Typing `/frpos {x} {y}` allows you to change the location. Screen positions are located as x-center (0 is center) and y-top (0 is the top of the screen) *note: If you cannot see the counter, reset the position to 0 0*
